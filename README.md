# NystromsHeroine

 Jump (Space) -> Diving (Down Arrow)

 Duck (Press Shift) -> Back to Standing (Release Shift)

 Protest (P) -> Start Revolution (R) -> March (M, control movement with WASD) -> Become King (K) -> Back to Standing (S)
